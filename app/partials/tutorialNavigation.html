Navigation

The main purpose of the navigation menu is to provide quick links to major sections of the application as well as the information of the current location.

It consists of the page title, Browse Stories and Browse Composites links.

  • Page title displays information about the current location (i.e. View Story) and links to the home screen.
  • Browse Stories opens the dialog that allows us to open an existing of create a new story.
  • Browse Composites opens the dialog that allows us to open an existing of create a new composites class.
