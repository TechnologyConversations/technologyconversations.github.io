Browse Composites

Composites are convenient way to define new steps composed of one or more predefined steps. Further on, composites are grouped into composite classes. Browse Composites dialog allows users to create new or open an existing composites class.

Click the Browse Composites in the navigation menu to open the dialog. Once inside, you can open any composites class. New composites class can be created by typing the name in the search field and clicking the Create New Composites Class () icon.
