Composites Form

Composites Form is a representation of a composites class. It is divided into 3 parts: Class Name, Composites and Composite Steps.

Class Name

Class Name is the unique identifier of a set of composites.

Composites

All composites are listed on the left-hand side of the screen. Buttons in this list can be used to add () and remove () composites. The search field can be used to filter composites.

Composite Steps

Each composite consists of one or more steps. Click on the desired composite (or create a new one) to open the composite form. Composite form consists of composite name and steps. Steps can be added (), moved () and removed (). Steps support the auto-complete feature. Start writing the word that identifies some step and the GUI will display the list of candidates. Once the desired step is found, space of enter key will add it to the corresponding field.
