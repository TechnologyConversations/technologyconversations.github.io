BDD Assistant Tutorial

Composites Operations

Following operations are allowed: create new composites, update existing composites, revert changes and delete composites.

Create new composites

If specified composites class does not exist, the "Create New Composites" button can be used.

Update composites

Existing composites can be updated using the "Update Composites" button.

Revert changes

Any change made to composites can be reverted to the version created on the last save. To revert composites, click the "Revert Changes" button. This operation cannot be undone. If there were no changes to composites, the button is disabled.

Delete composites

Any composites class can be deleted. This operation cannot be undone. If the composites class was not previously saved on the server, the button is disabled.
