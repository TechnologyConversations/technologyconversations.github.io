Multiple Stories Runner

Multiple Stories Runner can be used to run multiple stories. Every time implementation of some feature is done, multiple stories runner can be used to verify whether the whole system works as expected.

Tour the run stories selector dialog

Run Stories top menu item opens a dialog that allows navigation between directories and stories selection. Stories or whole directories with stories can be selected by clicking the checkbox on the right side. After the desired stories are selected, Continue button leads to the Runner Parameters dialog described in the "Run Story parameters dialog" section of the Single Story runner tutorial.

Once all stories are executed, the "Open the report >" link will appear providing all the information related to the execution of scenarios.
