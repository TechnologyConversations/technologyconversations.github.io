Navigation

Tour the navigation in the demo application

The main purpose of the navigation menu is to provide quick links to major sections of the application as well as the information about the current location.

Navigation menu consists of the page title, Browse Stories, Browse Composites and Run Stories.

  • Page title displays information about the current location (i.e. View Story) and links to the home screen.
  • Browse Stories displays the dialog that can be used to open an existing or create a new story.
  • Browse Composites displays the dialog that can be used to open an existing or create a new composites class.
  • Run Stories displays the dialog that can be used to select one or more stories and directories with stories that will be run.
