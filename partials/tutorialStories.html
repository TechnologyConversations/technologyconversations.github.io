Browse Stories

Tour the navigation in the demo application

Browse Stories dialog allows users to navigate directories and stories, create new directories, create new stories, open or delete an existing story.

Click the Browse Stories in the navigation menu to open the dialog. Once inside, you can navigate through directories () and open any story (). New directory can be created by typing the name in the search field and clicking the Create New Directory () icon. Create New Story allows story creation in the selected directory. Each story is followed with the Delete Story () icon.
